# The Study Lounge
The Study Lounge is a computer science focused study platform for SC, HSC and BSc computer science students. It provides structured, well-written notes organised by chapter along with a community Q&A system where students can ask questions and get help.

> Coming soon

## Features
### Notes Platform
- Browse notes by level (SC, HSC, BSc Computer Science)
- Notes organised by chapter (SC/HSC) or module (BSc)
- Search and filter notes

### Community Q&A
- Ask questions on any note
- Site-wide Q&A board
